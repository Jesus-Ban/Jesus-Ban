# Jesus Ban — Portfolio

> Personal portfolio website built with React + Vite.  
> Live at: **[jesusban.vercel.app](https://jesusban.vercel.app)**

---

## Tech Stack

- **React 18** — UI components
- **Vite** — build tool & dev server
- **Plain CSS-in-JS** — no Tailwind, no UI library
- **Google Fonts** — Syne, DM Sans, JetBrains Mono

## Features

- Smooth scroll navigation with active section tracking
- Scroll-triggered fade-in animations
- Fully responsive — mobile-first with hamburger menu
- Sections: About · Skills · Experience · Projects · Education · Contact

## Getting Started

```bash
# Install dependencies
npm install

# Run locally
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
├── Portfolio.jsx   # Main component (all sections)
├── main.jsx        # React entry point
└── index.css       # Global reset
```

## Deployment

Deployed on **Vercel**. Every push to `main` triggers an automatic redeploy.
